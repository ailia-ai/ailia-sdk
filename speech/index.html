ailia Speech

A library for speech recognition. It uses ailia SDK and ailia.audio for doing Speech-to-Text.

Getting Started

Run your first speech-to-text inference in three steps. See the API references below for C++, Unity, and Flutter.

1

Install

Install the ailia Speech Python package from PyPI.

pip3 install ailia_speech
View on PyPI
2

Run a Sample

Transcribe an audio file using a Whisper model. The model files are downloaded automatically into ./models/.

import ailia_speech
import librosa

audio, rate = librosa.load("demo.wav", mono=True)
speech = ailia_speech.Whisper()
speech.initialize_model(
    model_path="./models/",
    model_type=ailia_speech.AILIA_SPEECH_MODEL_TYPE_WHISPER_MULTILINGUAL_LARGE_V3_TURBO,
)
print(speech.transcribe(audio, rate))
