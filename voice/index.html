ailia Voice

A library for generating speech from text. Easily integrate AI powered text-to-speech into your applications.

Getting Started

Synthesize your first voice clip in three steps. See the API references below for C++, Unity, and Flutter.

1

Install

Install the ailia Voice Python package from PyPI.

pip3 install ailia_voice
View on PyPI
2

Run a Sample

GPT-SoVITS clones a target voice from a short reference clip. Set the reference audio, then synthesize. Models are downloaded into ./models/ automatically on first run.

import ailia_voice, librosa, soundfile

ref_text = "水をマレーシアから買わなくてはならない。"
ref_audio, rate = librosa.load("reference.wav", mono=True)

voice = ailia_voice.GPTSoVITS()
voice.initialize_model(model_path="./models/")
voice.set_reference_audio(
    ref_text, ailia_voice.AILIA_VOICE_G2P_TYPE_GPT_SOVITS_JA, ref_audio, rate,
)
buf, rate = voice.synthesize_voice(
    "こんにちは。今日はいい天気ですね。",
    ailia_voice.AILIA_VOICE_G2P_TYPE_GPT_SOVITS_JA,
)
soundfile.write("output.wav", buf, rate)

FAQ

Common questions about ailia Voice.

Which TTS models are supported?

ailia Voice currently bundles GPT-SoVITS, a zero-shot voice cloning model that synthesizes speech in the timbre of a short reference clip you provide.

What does "reference audio" mean and why is it required?

GPT-SoVITS clones the voice characteristics of a target speaker from a few seconds of clean reference audio plus the matching transcript. Pass both to set_reference_audio() before calling synthesize_voice().

Without reference audio, synthesis cannot start.

Which languages can I synthesize?

Japanese, English, and Chinese, selected via the AILIA_VOICE_G2P_TYPE_GPT_SOVITS_JA / _EN / _ZH constants passed to set_reference_audio() and synthesize_voice().

Can I use a custom pronunciation dictionary?

Yes. Pass a user_dict_path to initialize_model() pointing at a pyopenjtalk-compatible user dictionary to override default pronunciations for proper nouns or domain-specific terms.

Can I run it offline?

Yes, after the first run. Model weights are downloaded into the directory passed to initialize_model(model_path=...) on first use, and the evaluation license is fetched automatically. Subsequent runs work without an internet connection.

How does licensing work?

An evaluation license is downloaded automatically at runtime, suitable for development and trial. For commercial deployment, request a production license. See the ailia license terms.
