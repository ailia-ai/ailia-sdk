Python 環境のセットアップ

ailia SDK を Python から使い始めるために、Python・pip・git をインストールします。お使いの OS を選んでください。

はじめに

ailia SDK を Python から使うには、Python（プログラムを実行する環境）、pip（ライブラリをインストールするツール）、git（サンプルを取得するツール）の 3 つが必要です。OS ごとに準備の手順が少し異なるため、下記から環境に合ったガイドを選んでください。

Windows

Python・pip・git に加えて、ailia SDK の動作に必要な Visual C++ 再頒布可能パッケージのインストールまでを解説します。

  • Windows セットアップガイド

Mac

macOS には Python が標準で用意されているため、主に git の準備から始めます。

  • Mac セットアップガイド

Linux

Ubuntu / Debian 系を例に、パッケージマネージャ (apt) で不足分をインストールします。

  • Linux セットアップガイド

セットアップ後

環境が整ったら pip3 install ailia で ailia SDK をインストールできます。使い方は ailia SDK のはじめに を参照してください。GPU で高速に動かす場合は CUDA Toolkit / cuDNN インストールガイド や Vulkan セットアップガイド もあわせてご覧ください。
